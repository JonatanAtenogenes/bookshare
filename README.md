# BookShare

BookShare is a mobile application developed using Flutter, designed to facilitate the sharing and
exchange of books among users. This project aims to create a community where users can lend, borrow,
or exchange books with ease.

---

## Features

### Completed Features

- **Book Exchange System**: Users can propose and accept book exchanges.
- **User Profiles**: Each user can maintain a profile with their details and a list of available
  books.
- **Book Details**: View detailed information about books, including the synopsis, condition, and
  images.

### Upcoming Features

- **Donations**: A system to donate books will be included in future releases. This feature will not
  be available in the first version.

### Known Limitations

- **Search and Filters**: While the UI for searching and filtering books is available, the
  functionality is not implemented yet.
- **Input Validations**: Some validations for input fields are still missing and will be added in a
  future update.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/JonatanAtenogenes/bookshare.git
   ```

2. Navigate to the project directory:
    ```bash
    cd bookshare
    ```

3. Install dependencies:
    ```bash
    flutter pub get
     ```

4. Run the app:
    ```bash
    flutter run
     ```
